# Spec: AI brainrot typing game

## Requirements

### Requirement: fresh phrase generation
The system shall generate packs of humorous brainrot phrases with AI.

### Requirement: typo punishment
The system shall emit an immediate loud audio cue when the player types an incorrect character.

### Requirement: typing metrics
The system shall display WPM, accuracy, timer, streak, and final score.

### Requirement: instant demo
The system shall work immediately without login or setup.
