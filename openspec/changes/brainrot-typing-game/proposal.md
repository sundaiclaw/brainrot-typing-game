# Proposal: Brainrot Typing Game

## Why
Typing tests are usually sterile and forgettable. This project turns the core typing-speed loop into a funny, high-energy challenge with AI-generated brainrot phrases and an immediate loud punishment noise on every typo.

## What changes
Build a browser game where players race through absurd AI-generated brainrot phrases, track WPM/accuracy/streak, and hear an intentionally obnoxious buzzer the instant they type the wrong next character.

## Success criteria
- Player can start a timed run instantly with no login.
- App generates fresh brainrot phrase packs with AI.
- Wrong key triggers a loud audio punishment immediately.
- UI shows timer, WPM, accuracy, streak, and score.
